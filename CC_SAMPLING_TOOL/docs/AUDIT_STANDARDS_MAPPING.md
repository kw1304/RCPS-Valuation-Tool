# 감사기준·K-IFRS 매핑

이 문서는 본 툴이 구현한 각 기능의 감사기준 및 학술 출처를 기록합니다.
툴이 제안하는 모든 값은 참고치이며, 최종 판단은 감사인의 전문적 판단에 따릅니다.

---

## 1. MUS 알고리즘

### 출처
- **AICPA Audit Guide: Audit Sampling (AAG-SAM)**, Chapter 4
- **ISA 530** (International Standard on Auditing 530 — Audit Sampling)
- **감사기준서 530** (한국공인회계사회, ISA 530 채택)

### 핵심 수식

**표본규모 결정:**
```
n = (모집단 총액 × 신뢰계수) / 수행 중요성
```

**표본간격:**
```
SI = 잔여 모집단 금액 / 최종 표본규모
```

**무작위 시작점**: 0 ~ SI 사이 균등 분포 난수

**MUS 선택**: 누적 금액 기준 [RS, RS+SI, RS+2SI, ...] 구간에 해당하는 거래처 선택

### 구현 위치
`src/domain/mus.py` — `run_mus()` 함수

### 출처

| 위험 수준 | 내부통제 신뢰 Y | 내부통제 신뢰 N |
|---|---|---|
| 유의적위험 (Significant Risk) | 1.4 | 2.31 |
| 중간위험 (Moderate Risk) | 1.0 | 1.4 |

이 계수는 오류 미발견 시(zero-error assumption) 단측 95% 신뢰구간 기준입니다.

## 5. 대체적 절차

### 출처
- **ISA 505 §12**: 확인서 미입수 시 감사인은 대체적 감사절차를 수행해야 한다.
- **감사기준서 505 §12**: 동일

### 대체적 절차 유형별 감사 근거

| 유형 | 감사 근거 |
|---|---|
| 후속 입금 확인 | 기말 이후 현금 수령 → 실재성·완전성 동시 충족 |
| 세금계산서 확인 | 거래 실재성 + 측정의 정확성 |
| 계약서 확인 | 거래 발생 근거 (권리·의무 확인) |
| 기타 (내부 문서) | 보조적 증거 — 단독 충분성 제한적 |

### 커버리지 판정 기준
| 커버리지 | 결론 | 근거 |
|---|---|---|
| ≥ 95% | 충분 | 실질적 전액 확인 |
| 50% ~ 95% | 부분 | 일부 미확인 — 위험 수준 판단 필요 |
| < 50% | 미해소 | 충분한 증거 미확보 — 추가절차 또는 왜곡 가능성 검토 |

이 기준치는 실무 관행 참고치이며, 감사인 판단에 따라 조정 가능합니다.

## 7. 감사인 판단과 툴 제안값의 분리

본 툴이 제안하는 모든 수치 (PM, Key item 기준금액, 신뢰계수, 표본규모 등)는:

1. 감사인의 전문적 판단을 **대체하지 않습니다**.
2. 감사기준서 530 §A4에 따라 표본규모는 "특정 방법에 의해 결정될 수 있으나 그 판단은 감사인의 것"임을 명시합니다.
3. 실제 감사보고서에 첨부되는 조서에는 툴 결과 외에 감사인의 검토 의견이 별도 기재되어야 합니다.
