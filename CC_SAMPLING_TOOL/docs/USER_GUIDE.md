# 채권채무조회서 MUS 표본추출 툴 — 사용자 가이드

> 감사기준서 530 (표본감사) 기반 MUS(Monetary Unit Sampling) 자동화 툴입니다.
> 이 툴은 **감사인의 판단을 보조**하는 도구이며, 표본 선택 결론은 감사인이 최종 책임집니다.

---

## 시작하기

### 환경 요구사항
- Python 3.10 이상
- 필요 패키지: `pip install -r requirements.txt`
- (선택) Tesseract OCR: 스캔 PDF 처리 시 필요

### 서버 실행
```bash
python -m flask --app api.app run --host 127.0.0.1 --port 8520
```
브라우저에서 `http://127.0.0.1:8520` 접속.

## 5단계 워크플로우

### Step 0: 프로젝트 생성
화면 상단의 **신규 프로젝트** 버튼을 클릭합니다.
- **회사명**: 감사 대상 회사명 (조서 헤더에 표시)
- **결산일**: 감사 대상 기간 종료일 (예: 2025-12-31)
- **채권·채무**: 채권만, 채무만, 또는 양쪽 모두

파일 업로드:
1. **거래처별 원장** (.xlsx): 채권 시트 + 채무 시트 포함
2. **재무제표** (.xlsx): 자산·부채 잔액이 있는 B/S
3. **특관자리스트** (.xlsx): 특수관계자 이름 목록 (선택)

> 시트명은 자동 감지됩니다 — "채권"/"매출원장"/"AR", "채무"/"매입원장"/"AP" 등 한국어·영문 모두 지원.

---

### Step 1: 표본 설계 (MUS 샘플링)

**주요 파라미터:**

| 파라미터 | 설명 | 일반적 값 |
|---|---|---|
| 수행 중요성 (PM) | 총자산 × 중요성 비율 × 배분율 | 총자산의 1~2% |
| 위험 수준 | 유의적위험 / 중간위험 / 낮은위험 | 유의적위험 |
| 내부통제 신뢰 | 통제위험 감소 여부 (Y/N) | Y |
| Key item 기준금액 비율 | PM 대비 Key item 추출 배율 | 위험·통제에 따라 자동결정 |
| 신뢰계수 | 허용오류 관련 계수 (CF) | 위험·통제에 따라 자동결정 |

**결과물:**
- Key item 목록 (PM × ratio 이상 잔액)
- Representative 표본 (MUS 구간추출)
- 특관자 강제 포함

---

### Step 2: 조회서 발송명단 생성
- 채권: Step 1에서 `final_sampled=True`인 거래처 자동 추출
- 회신 기한, 감사인 연락처 입력 후 Excel 다운로드
- 발송은 감사인이 직접 처리 (이메일/우편)

---

### Step 3: 조서 생성 (Excel)
- 작성자·검토자 이름 입력
- **C100** (채권) 또는 **AA100** (채무) 조서 다운로드
- 포함 시트: Control Sheet / 표본규모 결정 / Key item / MUS 추출

---

### Step 4: 조회서 회신 처리
- PDF 회신 파일 일괄 업로드 (최대 수십 건)
- 자동 처리:
  1. PDF 텍스트 추출 (pdfplumber / Tesseract OCR)
  2. 거래처명 매칭 (퍼지 매칭)
  3. 회신금액 vs 장부가 비교 → 일치/불일치 판정
- 불일치 건: 수동 보정 UI 제공

> **Tesseract 미설치 시**: 텍스트 레이어 없는 스캔 PDF는 추출 불가. 수동 입력 필요.

---

### Step 5: 대체적 절차 및 증빙 집계
미회신 또는 불일치 거래처에 대해:
- **대체적 절차 유형**: 후속입금, 세금계산서, 계약서, 기타 확인
- **증빙 파일 업로드**: 여러 파일 합산
- **커버리지 판정**:
  - ≥ 95% → 충분
  - ≥ 50% → 부분
  - < 50% → 미해소
- 조서 최종 다운로드 (대체적 절차 시트 포함)

## 자주 묻는 질문 (FAQ)

### Q1. 왜 채권과 채무를 따로 처리하나요?
채권(AR)과 채무(AP)는 조회 대상, 잔액 부호, 조서 형식이 모두 다릅니다.
- 채권: 거래처에게 "우리 장부상 귀사가 X원을 우리에게 지불해야 한다"고 확인 요청
- 채무: 거래처에게 "우리 장부상 우리가 귀사에게 X원을 지불해야 한다"고 확인 요청

ISA 505 / 감사기준서 505에서 외부조회 목적과 방법이 채권·채무별로 별도 규정됩니다.

### Q2. MUS 표본간격(Sampling Interval)이란?
MUS는 모집단을 통화 단위로 나눕니다.

```
표본간격 = 잔여 모집단 금액 / 최종 표본규모
```

예) 잔여 모집단 100억, 표본규모 10 → 표본간격 10억

무작위 시작점(0 ~ 표본간격)을 정한 후, 누적 금액 기준으로 일정 간격마다 거래처를 선택합니다.
이 방법은 잔액이 큰 거래처가 선택될 확률을 높입니다 (확률비례추출).

### Q3. Key item 기준금액을 어떻게 결정하나요?
기준금액 = PM × Key item 비율

Key item 비율 매트릭스 (감사기준서 530 / AICPA AAG-SAM 기반):

| 위험 수준 | 통제 신뢰 Y | 통제 신뢰 N |
|---|---|---|
| 유의적위험 | 75% | 50% |
| 중간위험 | 100% | 75% |
| 낮은위험 | 150% | 100% |

툴이 제안하는 값은 감사인 판단의 참고치입니다. 직접 입력으로 재정의 가능합니다.

### Q4. 특관자는 왜 강제 포함되나요?
ISA 550 (특수관계자) / 감사기준서 550에 따라, 특수관계자 거래는 독립거래와 동일 조건 여부를 별도 검증해야 합니다. 금액 규모와 무관하게 조회서를 발송하는 것이 원칙입니다.

### Q5. 신뢰계수(CF)는 어떻게 결정되나요?
신뢰계수는 "표본에서 오류가 발견되지 않을 때 모집단 전체 오류 금액이 PM 이하"라는 감사 결론을 내리기 위한 통계적 계수입니다.

| 위험 수준 | 통제 신뢰 Y | 통제 신뢰 N |
|---|---|---|
| 유의적위험 | 1.4 | 2.31 |
| 중간위험 | 1.0 | 1.4 |
| 낮은위험 | 0.7 | 1.0 |

출처: AICPA AAG-SAM (Audit Sampling Audit Guide) Table 4-1.

### Q6. 회신 금액이 장부가와 다를 때 어떻게 되나요?
자동 판정 기준:
- |장부가 - 회신금액| ≤ 허용 오차(기본값 0원) → **일치**
- |장부가 - 회신금액| > 허용 오차 → **불일치 (mismatch)**

불일치 건은 Step 5 대체적 절차로 이동합니다. 감사인이 차이 원인을 소명하고 증빙을 첨부합니다.

### Q7. 조서 양식을 다른 회계법인 형식으로 바꿀 수 있나요?
`docs/TEMPLATE_REGISTRY.md`를 참조하세요. `configs/templates/` 폴더에 YAML 파일을 추가하면 됩니다.
