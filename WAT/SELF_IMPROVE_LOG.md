**발견된 결함 2건 (둘 다 fast 모드):**
1. **미검증 URL 환각** — 리스 손상 답변이 검색도 안 하고 `ifrs.org/.../leases-below-market-terms/`
   링크를 생성. 죽은/틀린 링크 위험.
2. **K-GAAP 정확성 오류(중대)** — 전환사채(일반기업회계기준)를 "전환권대가 분리 안 함,
   전액 부채"로 답함. **정밀검색 교차검증 결과 = 부채·자본 분리(전환권조정·전환권대가),
   전액 부채는 기준 위반.** 조문도 오인용(제6장 → 실제 제10장 10.7~10.12).
   → fast 모드의 지식기반 답변이 K-GAAP 세부에서 자신있게 틀림.

**조치:**
- fast 시스템 프롬프트 강화: ① 검색 안 했으면 출처 URL 임의 생성 금지(링크는 정밀검색 안내),
  ② 조문·분개·계정은 기억 의존이라 틀릴 수 있음 — 특히 K-GAAP 세부는 단정 말고
  '🔍정밀검색 확인 권고' 표시, K-IFRS↔K-GAAP 차이 명확히 구분.
- 프론트: fast 답변 하단에 **결정적 무결성 배지**(모델 의존 X) —
  "⚡ 빠른답변·지식기반(미검증). 조문·결론 확정 전 🔍정밀검색 교차확인 권고."

**검증:** 동일 전환사채 질문 재프로빙 → URL 환각 없음·정밀검색 권고 추가·
